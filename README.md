## Overview

This repository contains a simple personal landing page.

It is a minimal, static website designed to present basic information, core skills, and contact links in a clean and straightforward way.

## Features

* Minimal and responsive design
* Lightweight and fast (no frameworks)
* Accessible and semantic HTML
* Subtle animations and modern UI

## Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript
